# IMGD 5010 &bull; Assignment 07: Shaders

### [YouTube - May not work](https://www.youtube.com/watch?v=7K_U7SoCh3s) | [VIDEO FILE HOSTED ON DRIVE](https://drive.google.com/file/d/1amLrUYys8UcvHZGiO5Iy_ED6wohjeyNx/view?usp=drive_link)

I made an homage to Laurie Anderson's O Superman (1981). I was inspired as shaders were introduced to class, the first image rendered was a blurry circle in the middle of the screen - a shader facsimile of Anderson's iconic projection circle in its namesake video. 

Approaching the assignment, I was committed to first learning - at the very least - how to manage images. The math functions are like a candy store to me. I didn't want to launch into using them without first understanding how shader language worked to begin with. For that, I could make lines and circles. Luckily, that works well for making this homage. I enjoyed making it. I hope to do it again someday.
